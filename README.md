# Tres Puntos - Cocina Puertorriqueña Contemporánea

## Sitio Web Profesional

Bienvenido a la presencia digital de **Tres Puntos**, un restaurante de cocina puertorriqueña contemporánea que fusiona tradición con innovación culinaria.

### 🍽️ Sobre Nosotros

En Tres Puntos celebramos la excelencia gastronómica puertorriqueña con técnicas modernas y ingredientes frescos de calidad premium.

### 📁 Estructura del Proyecto

```
├── index.html              # Página principal
├── css/
│   ├── styles.css         # Estilos principales
│   ├── accessibility.css  # Estilos de accesibilidad
│   └── responsive.css     # Estilos responsive
├── js/
│   ├── main.js           # JavaScript principal
│   ├── menu.js           # Funcionalidad del menú
│   └── accessibility.js  # Mejoras de accesibilidad
├── assets/
│   ├── images/           # Imágenes del restaurante
│   ├── icons/            # Iconografía
│   └── fonts/            # Fuentes tipográficas
└── pages/
    ├── menu.html         # Página de menú
    ├── about.html        # Acerca de nosotros
    ├── reservations.html # Sistema de reservaciones
    ├── contact.html      # Contacto

```
